# AI Career Copilot

A Multi-Agent Placement Preparation & Career Intelligence Platform built using:

* LangGraph
* FastAPI
* Gemini
* RAG
* PostgreSQL
* FAISS
* Docker

## Features

* Resume Analysis
* Job Description Analysis
* Skill Gap Detection
* Learning Roadmap Generation
* Project Recommendations
* Mock Interview Preparation
